# Kauna Healthy

A fast, mobile-first BMI and ideal body weight calculator targeting Indonesian users. Bilingual (ID/EN) with SEO-optimized content in Bahasa Indonesia.

## Features

- BMI calculator with category breakdown (WHO standard)
- Ideal weight range via Devine formula
- Visual BMI gauge
- Bilingual — switch between Bahasa Indonesia and English instantly
- Mobile-first, responsive layout
- JSON-LD schema markup for rich search results

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
